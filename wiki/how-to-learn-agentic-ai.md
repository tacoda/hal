# How to Learn Agentic AI

A roadmap mind-map (Naresh Hingorani) to understand, build, and operate intelligent AI
agents — "start simple, build iteratively, ship responsibly." Agents don't just
perceive/reason/act — they act *toward goals*.

Branches around an **Agentic AI Basics** core:

1. **Core agent components** — profile/identity, memory (short & long-term, vector DB),
   planning & reasoning, tools & actions, observation/input, feedback loop; plus
   autonomy, perception, action, goal-orientation, reasoning & planning, learning.
2. **Agent frameworks & tools** — frameworks (LangChain, LlamaIndex, AutoGen, CrewAI,
   Microsoft AutoGen), model providers, tool integrations, vector stores, observability
   (LangSmith, Arize Phoenix, Helicone, TruLens, W&B), UI/interfaces.
3. **Key agentic capabilities** — task decomposition, tool use, multi-step reasoning,
   memory & context management, collaboration, adaptation & learning.
4. **Safety & governance** — safety & alignment, guardrails, content filtering,
   human-in-the-loop, auditing & logging, compliance & privacy.
5. **Agentic DevOps (AgentOps)** — monitoring, tracing, evaluation, alerting, cost
   management.
6. **Versioning & model management** — model versioning, prompt management,
   experimentation, rollouts, registry.

**Foundational knowledge:** LLMs & prompting, RAG, APIs/functions, data structures &
databases, Python, system design, AI safety & ethics.

**Suggested path:** learn the basics → build a simple agent → add memory & tools → add
guardrails → productionize → iterate & scale. **Golden tips:** build small & iterate,
observe/measure/learn from failures, safety & trust first, automate what can be
automated, focus on real user value not demos.
