# Build Your First Claude Automation

A 3-level roadmap (GenAI.works) from one command to full autopilot.

1. **Skills in Claude Chat** — a straight line you run by hand, start to finish. Save a
   task into a Skill (`.md`), fire the whole workflow with one trigger word. Steps:
   describe it once (Claude writes the `.md` skill), connect your tools (Notion, Gmail,
   Drive), test/fix/save. Example: type `/reel` → skill runs → output saved to calendar.
2. **Scheduled in Cowork** — a loop on a timer; it reruns itself with no hands. Hand the
   same skill to Cowork and tell it what to run and when (`run my skill, 9am` — no code).
   Answer a few setup questions, then it runs itself; result waits for your review.
3. **Claude Code** — a cloud runner that fans out to every tool you have. Move the same
   skill onto Claude Code so it runs in the cloud, reaching any tool at any hour. Install
   once, `/schedule` a time, walk away (laptop closed); manage runs from the browser.
   It fans out to your files, any API, and agents.

The through-line: the *same* skill graduates from manual → scheduled → cloud-autonomous
without rewriting it.
