# UX & Design

Design & usability — DOET, Refactoring UI, mobile interfaces, analytics.

- [Color Psychology](color-psychology.md) — what each color communicates, by emotion/industry/brand
- [Designing for Performance](designing-for-performance.md) — treat page speed as a design decision: aesthetics vs bytes, a performance budget, speed as a feature
- [Designing Mobile Interfaces](designing-mobile-interfaces.md) — Hoober & Berkman's device-agnostic library of 70+ mobile UI interaction patterns
