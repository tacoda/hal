# Color Psychology: What Colors Communicate

A reference table (Ashley Couto) mapping colors to the emotion they evoke, the
industries that use them, what they're used to do, and example brands.

| Color | Emotion | Used to | Example brands |
|---|---|---|---|
| Red | Excitement, energy, passion, courage | Stimulate, create urgency, draw attention | Coca-Cola, Netflix, Target |
| Orange | Optimistic, independent, adventurous | Stimulate, communicate fun, draw attention | Nickelodeon, Fanta, Home Depot |
| Yellow | Enthusiasm, originality, happiness | Stimulate, encourage, awaken, create hope | McDonald's, Snapchat, IKEA |
| Teal | Growth, health, fertility, dependability | Restore energy, promote growth, rejuvenate | Whole Foods, Spotify, Tiffany & Co. |
| Green | Safety, harmony, stability, balance | Relax, balance, revitalize, encourage | Starbucks, Animal Planet, John Deere |
| Sky Blue | Freedom, self-expression, trustworth, joy | Draw attention, inspire trust, communicate | Twitter, LinkedIn, Intel |
| Royal Blue | Trust, responsibility, honesty, loyalty | Reduce stress, create calmness, secure | Facebook, Ford, American Express |
| Violet | Imagination, spirituality, compassion | Encourage creativity, combine wisdom & power | Yahoo!, Twitch, Hallmark |
| Pink | Compassion, love, playful, admiration | Communicate energy, motivate action | T-Mobile, Barbie, Victoria's Secret |
| Brown | Reliability, stability, honesty, comfort | Imply common sense, support emotions, warmth | UPS, Hershey's, Louis Vuitton |
| Gray | Neutral, practical, conservative, formal | Create composure, timeless, mature | Apple, Wikipedia, NYT |
| Black | Power, control, authority, elegance | Hide feelings, intimidate, radiate authority | Nike, Chanel, Adidas |
