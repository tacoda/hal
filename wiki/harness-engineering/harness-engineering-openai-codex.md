# Harness Engineering with Codex (OpenAI)

Ryan Lopopolo's OpenAI essay is the most-cited artifact in the harness-engineering
wave: an internal team spent **five months building and shipping a beta software
product with more than a million lines of code and zero lines written by hand** —
every line generated by Codex agents. It's the concrete existence proof behind
the *"2026 is agent harnesses"* claim (see
[2025 Was Agents, 2026 Is Agent Harnesses](2025-agents-2026-agent-harnesses.md))
and the practitioner detail is expanded in the interview,
[Extreme Harness Engineering for Token Billionaires](extreme-harness-engineering-token-billionaires.md).

## What "agent-generated" actually means

Everything in the codebase is agent-produced — not just product code:

- Product code **and** tests
- CI configuration and release tooling
- Internal developer tools
- Documentation and design history
- **Evaluation harnesses** (the agents build the very checks that grade them)
- Review comments and responses
- Scripts that manage the repository itself
- Production dashboard definitions

## Humans move up an abstraction layer

Humans stay in the loop but operate differently: they **prioritize work,
translate user feedback into acceptance criteria, and validate outcomes** rather
than write code. Agents use the team's standard dev tools directly — pulling
review feedback, responding inline, pushing updates, and often squashing and
merging their own PRs.

The central discipline matches Mitchell Hashimoto's *engineer the harness* step
(see [My AI Adoption Journey](../agentic-coding/my-ai-adoption-journey.md)): **when the agent
struggles, treat it as a signal.** Identify what's missing — a tool, a guardrail,
documentation — and feed the fix back into the repository, *always by having
Codex itself write the fix.* This is the [self-improving harness loop](self-improving-harness-loop.md)
run at industrial scale, and it depends on the machine-checkable
[sensors](ai-coding-sensors.md) and simulators from
[harness-engineering](harness-engineering.md).
